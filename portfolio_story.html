Chapter 02 — Who I Am

The Story
So Far

"I believe great ML engineering is
90% about the data and

I'm Parth Tyagi, a B.Tech student in Mathematics & Computing at Central University of Karnataka. My obsession is building machine learning systems that are honest, robust, and actually deployable.

What sets my work apart isn't just model accuracy — it's the discipline I apply before the model even trains. Proper outlier treatment, thoughtful feature engineering, clean preprocessing pipelines. I've learned the hard way that a leaky pipeline produces a lying model.

I've shipped three live AI applications — each one a complete story: a problem worth solving, data cleaned with care, a model chosen for good reasons, and a UI that a real person can actually use.

Currently expanding into backend API integration, scalable ML systems, and LLM fine-tuning — eyes set firmly on an ML engineering role in 2026.
